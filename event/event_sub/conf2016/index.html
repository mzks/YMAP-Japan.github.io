「第1回 宇宙素粒子若手の会 秋の研究会」は
2016年10月29日(土)-31日(月)に東京大学 柏キャンパス（千葉県柏市）で開催されます。

私たち宇宙素粒子若手の会では、同年代の若手研究者と宇宙線分野全体にわたる様々な研究について議論し、 交流を深める機会として、宇宙素粒子分野全体および光検出器開発をカバーする学生・若手研究者による研究会の開催を行います。
本研究会は参加者全員による口頭発表の形式をとります。
宇宙線分野に携わる・若しくは興味のある学生・若手研究者の皆さまは是非ともご参加ください。
若手の会一同、こころよりお待ちしております。

参加資格

広義の宇宙線研究に関心がある学生・若手研究者が対象となっています。
事前にレジストレーションフォームからアブストラクトを提出して下さい。

招待講演

今回は宇宙素粒子の分野から2名の招待講師をお招きする運びとなりました。
実験分野からは伊藤洋介さん（東京大学ビッグバン宇宙国際研究センター）、
理論分野からは伊部昌宏さん（東京大学宇宙線研究所）です。
伊藤さんの専門分野は重力波の直接探索であり、最近では直接観測が成功したことにより耳にした人も多いのではないでしょうか。
伊部さんの専門分野は暗黒物質であり、普段は探索実験の話をよく聞きますが、今回は理論の側面から講演していただきます。
招待講演は初日（10/29(土)）の午後のセッションにて予定しています。
